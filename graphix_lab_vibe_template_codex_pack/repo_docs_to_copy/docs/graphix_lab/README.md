# Graphix Lab documentation

Graphix Lab is an educational usability layer over Graphix for measurement-based quantum computing.

It focuses on:

- concise wrappers around Graphix circuits and patterns
- readable command tables and resource summaries
- simulation reports that delegate to Graphix backends
- conceptual traces of MBQC command execution
- Matplotlib/NetworkX visualizations
- optional Qiskit import for a small, well-tested gate subset

It does not attempt to replace Graphix internals.
