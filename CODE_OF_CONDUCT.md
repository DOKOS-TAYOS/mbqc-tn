# Code of Conduct

This project follows a simple expectation: be respectful, constructive, and collaborative.

## Our Standards

- Assume good intent and communicate with clarity.
- Focus feedback on code, behavior, and outcomes instead of people.
- Make space for questions from contributors with different levels of experience.
- Avoid harassment, insults, discrimination, and intimidation.

## Scope

These expectations apply to discussions, issues, pull requests, chats, demos, and any other project spaces.

## Reporting

Report concerns privately to the maintainers listed by the project after bootstrap. Until then, treat this file as the template baseline to customize.
